# SQL Analytics Playbook

A generic repository for shareable SQL patterns, analysis snippets and real-world data scenarios, designed to demonstrate end-to-end analytical thinking, data modeling skills, and business case execution

## Structure
```
/projects/<topic>/      # Self-contained examples with context & docs
/docs/                  # Generic documentation (schemas, conventions)
/src/sql/               # Reusable SQL patterns (CTEs, utilities, macros-in-SQL, etc.)
```
## Analytical Methodology
| Step                    | Description                                       |
| ----------------------- | ------------------------------------------------- |
| Define business problem | Understand what we need to solve & why it matters |
| Explore data assets     | Identify tables, keys, metrics & grain            |
| Design logic            | CTEs, joins, ranking logic, window functions      |
| Build SQL model         | Efficient, readable, resilient queries            |
| Validate                | QA, data consistency checks, edge cases           |
| Translate insights      | Explain logic in business terms                   |
